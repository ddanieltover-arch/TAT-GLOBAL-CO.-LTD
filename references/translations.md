## Notes on Thai Language Implementation

1. **Font:** Use `Sarabun` or `Noto Sans Thai` for Thai text. Add to `next/font/google` alongside Playfair Display and DM Sans.

2. **Text sizing:** Thai script often needs 1–2px more line-height than Latin. Apply `leading-relaxed` to Thai body text.

3. **Language switcher implementation:**
   ```tsx
   // Simple locale link approach with next-intl
   import { useLocale } from 'next-intl'
   import { Link } from '@/i18n/navigation'
   
   const locale = useLocale() // 'en' or 'th'
   const otherLocale = locale === 'en' ? 'th' : 'en'
   const otherLabel = locale === 'en' ? 'ภาษาไทย' : 'English'
   ```

4. **SEO for Thai pages:**
   - `hreflang="th"` on Thai pages
   - Thai keyword: "ผู้ส่งออกข้าวไทย", "ซัพพลายเออร์ข้าวหอมมะลิ", "บริษัทส่งออกข้าวประเทศไทย"
